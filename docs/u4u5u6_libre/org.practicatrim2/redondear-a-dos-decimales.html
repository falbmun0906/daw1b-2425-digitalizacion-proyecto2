fun <T : Number> redondearADosDecimales(valor: T): Double

Redondea un número a dos decimales.

Esta función toma un valor numérico genérico (ya sea de tipo Double o Float) y lo redondea a dos decimales. Si el valor no es de un tipo soportado (es decir, si no es Double ni Float), lanzará una excepción IllegalArgumentException.

Return

El valor redondeado a dos decimales como un Double.

Parameters

valor

El valor numérico que se desea redondear. Este puede ser de tipo Double o Float.

Throws

IllegalArgumentException

Si el tipo de valor no es soportado.

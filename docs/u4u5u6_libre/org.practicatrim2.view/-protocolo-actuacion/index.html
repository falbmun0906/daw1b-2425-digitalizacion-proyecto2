ProtocoloActuacion

@Serializable
data class ProtocoloActuacion(val fallo: String, val protocolo: String, val descripcion: String, val tiempo_estimado: String? = null, val herramientas_necesarias: List<String> = emptyList(), val acciones: List<String> = emptyList())

Representa un protocolo de actuación en caso de fallo de un componente de un brazo robótico. Contiene la descripción del fallo, el protocolo a seguir, herramientas necesarias, acciones a tomar, y el tiempo estimado para realizar el protocolo.

Parameters

fallo

Descripción del fallo detectado en el sistema (e.g., "Fallo en el motor", "Sensor de temperatura defectuoso").

protocolo

Nombre o identificador del protocolo a seguir ante el fallo.

descripcion

Descripción detallada de lo que implica el protocolo de actuación.

tiempo_estimado

Tiempo estimado para completar el protocolo (puede ser nulo).

herramientas_necesarias

Lista de herramientas necesarias para realizar el protocolo (por defecto está vacío).

acciones

Lista de acciones que deben realizarse para solucionar el fallo (por defecto está vacío).
